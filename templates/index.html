iRAP UI is a web interface designed to interact with the iRAP pipeline. iRAP is a flexible computational pipeline for RNA-seq that integrates many existing tools for filtering and mapping reads, quantifying expression and testing for differential expression. It is aimed at a broad group of users with basic bioinformatics training and requires little experience with the command line but also provides more advanced users with the ability to customise the options used by their chosen tools.
